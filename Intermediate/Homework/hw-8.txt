Problem 1: Modify the Calculator
Using the code provided in class, add a multiplication operation to the calculator. Follow these steps:
- Create a new function called multiply that takes two arguments and returns their product.
- Update the calculate function to include multiplication when the user selects the "*" operation from the dropdown.
- Add "*" to the dropdown menu of operations.
Hint: Remember to handle any potential errors.


Problem 2: Create a Division Function
- Write a function named divide that takes two numbers and returns the result of dividing the first number by the second.
- Update the calculate function to include division when the user selects the "/" operation.
- Add "/" to the dropdown menu of operations. Make sure to handle division by zero!


Problem 3: Customizing the GUI

Problem 4: Create a Reset Function

- Update the "Reset" button to call this new function instead of the clear function.


Problem 5: User Experience Improvement
